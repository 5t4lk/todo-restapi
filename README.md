# REST API for creating TODO Lists

## Project includes:
- Developing Web Application following to design of REST API.
- Working with framework <a href="https://github.com/gin-gonic/gin">gin-gonic/gin</a>.
- The Pure Architecture approach to building an application structure. Dependency implementation techniques.
- Working with MongoDB.
- Configuring application with help of library <a href="https://github.com/spf13/viper">spf13/viper</a>. Working with environment variables.
- Registration and authentication. JWT. Middleware.
- Writing NoSQL queries.
- Graceful Shutdown.
- Docker

### To run application:

```
make run
```
